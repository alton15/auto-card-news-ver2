# Learn

현재 세션에서 재사용 가능한 패턴을 추출합니다.

## Process

### 1. Review Session
- 이번 세션에서 수행한 작업 검토
- 반복되는 패턴 식별
- 발견한 문제와 해결책 정리

### 2. Extract Patterns
- **코딩 패턴**: 자주 사용된 코드 구조
- **디버깅 패턴**: 에러 해결 방법
- **아키텍처 결정**: 설계 선택과 이유

### 3. Update Documentation
- CLAUDE.md에 새로운 규칙 추가
- AGENT_HANDOFF.md에 새로운 이슈/해결책 추가
- rules/ 파일 업데이트

## Output Format

```markdown
## 학습 내용

### 새로운 패턴
- [패턴 설명]

### 발견된 이슈
- [이슈 + 해결책]

### 규칙 업데이트
- [추가/변경할 규칙]
```
